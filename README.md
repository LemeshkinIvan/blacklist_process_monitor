# blacklist_process_monitor

Небольшой PowerShell-скрипт для мониторинга и блокировки нежелательных процессов на Windows.

## Описание

Скрипт работает в бесконечном цикле с заданным интервалом (default — 5 секунд) и выполняет следующие действия:
- сканирует список запущенных процессов
- ищет процессы по совпадению имени с blacklist’ом
- при обнаружении запрещённого процесса:
  - получает информацию о процессе:
    - имя
    - PID
    - пользователь, запустивший процесс
    - путь к исполняемому файлу
  - завершает процесс и все его дочерние процессы
  - логирует все действия
- после обработки возвращается к повторному поиску

## ⚙️ Конфигурация

Скрипт использует JSON-файл конфигурации (`config.json`), в котором задаются:

- `processBlacklist` — список запрещённых процессов (указывать **без `.exe`**)
- `logDirectory` — директория для логов
- `timeSleepSeconds` — интервал между проверками в секундах

Пример:

```json
{
  "processBlacklist": [
    "draw.io",
    "Telegram"
  ],
  "logDirectory": "",
  "timeSleepSeconds": 5
}
```
## В планах
- Запись логов в файл (помимо вывода в консоль)
- Скачивание конфигурации с удалённого устройства по HTTP
  - Загрузка будет выполняться автоматически с учётом интервала, указанного в конфиге
